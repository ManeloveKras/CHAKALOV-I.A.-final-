# Автоматизированный мониторинг и реагирование на угрозы

Итоговое домашнее задание по курсу "Программирование на Python".

## Функционал скрипта
- **2 источника данных**: API Vulners + имитация логов Suricata
- **Анализ**: IP с трафиком >100 запросов, CVSS >7.0
- **Реагирование**: Консольные уведомления о блокировке IP и обновлении ПО
- **Отчеты**: `threat_report.json`, `threat_report.csv`
- **Визуализация**: `threats_plot.png` (события логов + гистограмма CVSS)
